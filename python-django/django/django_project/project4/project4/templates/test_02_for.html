กรณีที่เป็นข้อมูลชนิด dict
แสดงทั้ง key, value
    {% for k, v in dict.items %}
    {{k}}: {{v}}
    {% endfor %}
แสดง key
    {% for k in dict.keys %}
    {{k}}
    {% endfor %}
แสดง value
    {% for v in dict.values %}
    {{v}}
    {% endfor %}
กรณีที่เป็นข้อมูลชนิด empty
    {% for x in empty_list %}

    {% endfor %}
กรณีที่เป็นข้อมูลชนิด cycle
    {% for _ in range %}
  • {% cycle 'One' 'Two' 'Three' 'Four' 'Five' %}
    • {% endfor %}
loop-แรกจะแสดงคำว่า One, loop-สองจะแสดงคำว่า Two แล้ว Three, Four ตามลำดับ
สำหรับตัวแปร forloop.counter
    {% for x in list %}
    รายการที่ {{ forloop.counter}}: {{x}}
    {% endfor %}
สำหรับตัวแปร forloop.counter0
    {% for x in list %}
    รายการที่ {{ forloop.counter0}}: {{x}}
    {% endfor %}
สำหรับตัวแปร forloop.revcounter
    {% for x in list %}
    รายการที่ {{ forloop.revcounter}}: {{x}}
    {% endfor %}
สำหรับตัวแปร forloop.revcounter0
    {% for x in list %}
    รายการที่ {{ forloop.revcounter0}}: {{x}}
    {% endfor %}
